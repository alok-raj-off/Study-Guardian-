# Yoriichi Study Pillar: Breath of Focus ⚔️

A high-intensity, character-driven productivity tracker inspired by the legendary **Yoriichi Tsugikuni** from *Demon Slayer*. This desktop application uses visual and auditory reinforcement to ensure your study sessions remain as disciplined as a Sun Breathing master.

---

## 🌟 Core Concept
This isn't just a timer; it's a presence on your screen. When you start your session, Yoriichi "awakens" with his iconic Japanese dub to motivate you. If you stay consistent, he honors your effort; if you fail your goal, you face his silent, fiery rage.

## ✨ Features
* **Always-on-Top Widget:** A frameless, draggable character that stays pinned to your workspace.
* **Dynamic Visual States:**
    * **Dormant:** Eyes closed, calm breathing.
    * **Awakened:** 10% to 100% eye-opening animation upon starting the timer.
    * **Focus Mode:** Subtle hair movements and blinking to keep the character "alive".
    * **Failure (Dishonor):** Intense scowl with glowing red eyes and a slow-draw katana animation.
    * **Success (Honor):** Calm parting dialogue with a "Good Luck" sign.
* **Japanese Dub Integration:** Audio triggers for session starts, reminders, and conclusions.
* **Adaptive Window:** The window automatically shrinks and moves to the corner of your screen when the session begins.

## 🛠️ Tech Stack
* **Language:** Python
* **GUI Framework:** PyQt6 (for frameless, translucent, and draggable window management)
* **Graphics Engine:** Pillow (PIL) for frame-by-state image handling
* **Audio Engine:** Pygame.mixer for high-fidelity voice line triggers

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/yourusername/yoriichi-study-pillar.git](https://github.com/yourusername/yoriichi-study-pillar.git)
   cd yoriichi-study-pillar
   ```

2. **install and set the setup:**
   ```bash
   npm i
   npm run setup
   ```

3. **Run the app**
   ```bash
   npm start
   ```
